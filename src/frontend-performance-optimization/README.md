### 浏览器
1. 作用：向服务器发出请求，在浏览器窗口中展示选择的网络资源。
   + 软件迭代更新的重要性（居安思危）
   + 用户体验、用户感知的重要性（人性化产品）
   + 互联网时代，什么都是可以替代的
   + 分享、免费、共筑共赢才是互联网的精神
2. 组成结构
   + 用户界面（User Interface）
   + 浏览器引擎（Browser engine）
   + 渲染引擎（Rendering engine） ▲
   + 网络（Networking）
   + 用户界面后端（UI Backend）
   + JavaScript解释器（JavaScript Interpreter）
   + 数据存储（Data Persistence）
3. 浏览器是多进程的
   + Browser进程
   + 第三方插件进程
   + CPU进程
   + 渲染进程
